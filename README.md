# 🍉 MELON MADNESS 67

Ein retro Arcade-Basketballspiel im Stil einer osteuropäischen Cartoon-Komödie.

## 🎮 Spielen

👉 **[Jetzt spielen](https://davidnikischin0601.github.io/WaterMelonGame67/)**

## Steuerung

| Aktion | Handy | PC |
|---|---|---|
| Wolf bewegen | ◀ ▶ Buttons | ← → / A D |
| Werfen | Einmal tippen | Klick / Leertaste |

## Punkte-Zonen

| Zone | Punkte |
|---|---|
| Weit (grün) | +3 |
| Normal (gelb) | +2 |
| Nah (rot) | +1 |

## ⚡ Der 67-Move

Der Hase wird riesig und tanzt, wenn der Score:
- die Ziffer **6** oder **7** enthält
- oder eine **Quersumme von 13** hat (weil 6+7=13)

## Technologie

Reines HTML5 Canvas + JavaScript — keine externen Abhängigkeiten.
